## 桌面快捷方式
```bash
[Desktop Entry]
Name=Clash for Windows
Comment=A Windows-style GUI for Clash
Exec="./cfw"  # 你的cfw可执行文件路径
Terminal=false  # 启动时不显示终端
Type=Application
Categories=Network;Utility; 
Icon="./cfw.png"  
StartupWMClass=Clash for Windows  
```
